# 随时记

移动端优先的网页 MVP：输入或语音说一句话，自动整理成待办、日程、提醒或备注，并保存到浏览器本地。

## 运行

```bash
npm install
npm run dev
```

打开：

```text
http://localhost:5173
```

## 当前功能

- 文字快速记录
- 浏览器语音输入，取决于浏览器支持情况
- 中文规则解析日期、时间、类型和提醒
- 确认后保存到本地 `localStorage`
- 按逾期、今天、明天、以后、无日期、已完成分组
- 搜索、完成、删除、延期到明天
- 页面打开期间支持浏览器通知提醒
- 记录、事项、每日总结三页分离
- 每日 AI 总结：按天缓存，页面打开时自动生成；页面持续打开时会在 0 点后刷新当天报告

## 下一步建议

1. 接入后端数据库和登录，避免换设备后数据丢失。
2. 部署后端定时任务，让网页关闭时也能在 0 点自动生成日报。
3. 接入邮件、企业微信或短信提醒，让网页关闭后也能可靠提醒。
4. 做成 PWA，方便添加到手机桌面。
